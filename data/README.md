# Data

MRI images are not included in this repo due to file size. Two datasets were used:

- **Binary dataset (Yes/No tumor):** https://www.kaggle.com/datasets/navoneel/brain-mri-images-for-brain-tumor-detection
- **Multi-class dataset (Glioma / Meningioma / Pituitary):** Source Lost, folder is too big to upload to GitHub — contains 6,056 images across glioma, meningioma, and pituitary tumor classes.

Download both and upload to Google Colab before running the notebook.
